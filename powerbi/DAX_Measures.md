# DAX Measures Documentation

---

# Overview

This document contains all DAX measures used in the Insurance Business Intelligence Dashboard.

The objective of these measures is to standardize business calculations and ensure consistency across all report pages.

Measures should always be preferred over calculated columns whenever aggregation is required.

---

# Measure Naming Convention

| Prefix | Purpose |
|---------|---------|
| Total | Overall aggregation |
| Avg | Average values |
| Count | Count calculations |
| Rank | Ranking |
| % | Percentage |
| Dynamic | Dynamic text |

Example

```
Total Premium
Total Commitment
Achievement %
Average Premium
```

---

# 1. Agent Actual

## Business Purpose

Calculates the total Net Premium generated by agents.

## Formula

```DAX
Agent Actual =
SUM(Fact_Business_Raw[NetPrem])
```

## Returns

Decimal

## Used In

- KPI Card
- RM Dashboard
- Agent Dashboard
- Company Analysis
- Product Analysis

## Validation

Compare with Pivot Table total.

Expected result should match the source report.

---

# 3. Achievement %

## Business Purpose

Measures business performance against assigned commitment.

## Formula

```DAX
Achievement % =
DIVIDE(
    [Agent Actual],
    [Agent Committed],
    0
)
```

## Returns

Percentage

## Interpretation

100%

Commitment Achieved

Greater than 100%

Over Achieved

Below 100%

Under Achieved

# Performance Recommendations

Always use Measures.

Avoid Calculated Columns unless row-level calculations are required.

Prefer DIVIDE() over "/".

Avoid nested IF statements.

Avoid repeated CALCULATE where unnecessary.

Use variables for complex measures.

---

Before publishing ensure:

✓ Percentages formatted correctly

✓ Slicers affect measures correctly
